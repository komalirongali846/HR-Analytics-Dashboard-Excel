# HR Analytics Dashboard

## Tools Used
- Microsoft Excel
- Pivot Tables
- Pivot Charts
- Slicers

## KPIs
- Total Employees
- Attrition Rate
- Average Salary
- Average Experience
- Average Performance

## Key Insights
- Finance has the largest workforce.
- Finance has the highest attrition count.
- Overall attrition rate is 19%.
- Senior employees form the largest experience group.
